== Description ==

This plugin adds an “Open In Social Debugger” link to the admin bar. The link opens the current page or post in Facebook Sharing Debugger.

This plugin works with pages and posts, and in both backend and frontend, as long as the admin bar is visible AND the page/post is in a published state.

### Usage

On any page or post, in frontend or backend, you'll see a "Open in Social Debugger" link in the admin bar, which opens the Facebook Sharing Debugger with the permalink of the current page/post for debugging.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= How do I use it? =

On any page or post, in frontend or backend, you'll see a "Open in Social Debugger" link in the admin bar, which opens the Facebook Sharing Debugger with the permalink of the current page/post for debugging.

= I can't see the link =

Make sure the admin bar is visible, and not hidden with css or a plugin which hides it. The post/page's status must also be published for the link to be visible.
